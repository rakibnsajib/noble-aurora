A comprehensive research project funded by TM RD (RM 252,000) focused on developing robust data quality control mechanisms for IoT-based smart home systems.

<!--more-->

This project addresses the critical challenge of ensuring data quality in IoT smart home environments where sensor data can be noisy, incomplete, or erroneous. The research developed machine learning-based approaches to automatically detect and clean dirty data in real-time IoT data streams, ensuring reliable system operations and accurate analytics.
